# Nutreetious

Nourish your dependency tree 🌱

A cross-platform Electron desktop application that visualizes and analyzes NuGet package dependencies in .NET projects. Quickly identify version conflicts, transitive dependencies, and dependency chains in your solutions.

## Features

- **Dependency Tree Visualization**: Interactive tree view of all NuGet package dependencies
- **Version Conflict Detection**: Automatically highlights version mismatches between requested and resolved packages
- **Multi-Project Support**: Analyze entire solutions or individual projects
- **Search & Filter**: Find packages quickly with real-time search highlighting
- **Framework Support**: View dependencies per target framework
- **Drag & Drop**: Load projects by dragging .sln, .slnx, or .csproj files
- **Transitive Dependencies**: Drill down into indirect dependencies
- **Cross-Platform**: Runs on Windows and Linux

## Installation

### Prerequisites

- Node.js 24+ (check [.github/workflows/build.yml](.github/workflows/build.yml) for current version)
- npm or your preferred package manager

### Setup

```bash
# Clone the repository
git clone https://github.com/ChristopherSteiner/nutreetious.git
cd nutreetious

# Install dependencies
npm install

# Start development server
npm run dev
```

## Known Limitations

Requires project.assets.json (generated by dotnet restore)
Supports .sln, .slnx, and .csproj files (not .vdproj or other project types)
Large projects with 1000+ transitive dependencies may experience performance lag
